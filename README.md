alfred-dissector
================

Simple Wireshark dissector prototyp for [A.L.F.R.E.D](http://www.open-mesh.org/projects/open-mesh/wiki/Alfred "Alfred Project")


Setup
-----
* Put alfred.lua in ~/.wireshark/plugins
* (re)start Wireshark
